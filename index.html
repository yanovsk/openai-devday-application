Developing with LLMs


Slack Knowledgebot for Pulley (YC20)

A slackbot that accepts users' queties about the company and responds using relevant slack history as a context.

  • Used GPT-3.5 to turn raw slack thread history into a set of questions and created vector embeddings for (using ada-2) those questions.
  • Used GPT-4 functions API feature to fill parameters to query Pinecone with (before and after timestamp, channel name)
  • Is actively used across all teams at Pulley (65 people, 10+ questions daily). Already saved hundreds of hours for Pulley.

Codebase Semantic Search (R&D for Pulley)

Build and internal semantic code search using GPT-4 and Vector Embeddings.

  • Split the codebase and for each chunk of code use GPT-3.5 to describe its intent and embed this description. Include all functions called from inside this code element.
  • Codebase is split using three scopes (low: by functions/methods, mid: by files, high: by packages). GPT-4-32k is then used to create a description and Q&A of each scope, where higher scope is injected as a context. So, if GPT is asked to describe the code of the function, the file of this function is injected as a context
  • Chunks are turned to vector embeddings using ada-2. After top-k relevant chunks are returned from vector db we inject their code and the code of the functions called from inside them to GPT-4-32k to give a more detailed context to answer the question.

Teaching Ukranian teens about GPT

Founded a tutition-free educational program ULTA focused on AI and GPT technologies for talented Ukrainian youth. After a competitive selection process, only 20 finalists were chosen from a pool of 725 applicants. These finalists underwent a two-week intensive course covering foundational and advanced aspects of AI and GPT. Upon course completion, participants were paired with mentors from MIT to further deepen their understanding and prepare them for a career in AI.
